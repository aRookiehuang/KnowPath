# 知途（KnowPath）作品说明书

## 一、作品名称
**知途（KnowPath）**

## 二、作品定位
知途是一款以“目标澄清、路径规划、对话学习、状态反馈”为核心的 AI 学习平台，面向那些“有任务、有目标，但暂时说不清楚如何推进”的学习者。

## 三、作品应用场景
1. **比赛项目学习与拆解**：在有限时间内完成竞赛或课程项目。
2. **自主学习与能力提升**：围绕具体成果来安排学习路径。
3. **AI 陪练场景**：在任务推进中获得追问、反馈和提示。
4. **教学辅助与过程追踪**：帮助教师或助教观察学习过程。

## 四、设计理念
### 1. 不是答题器，而是学习工作台
知途强调围绕任务推进组织学习，而不是只做单次聊天问答。

### 2. 从模糊目标出发
许多学习问题不是“不懂某个点”，而是“不知道自己到底该解决什么问题”，所以目标澄清被放在最前面。

### 3. 对话是组织学习的方式
对话用于理解、拆解、引导和反馈，不只是生成内容。

### 4. 强调沉浸与清晰
前端统一中文、统一品牌、统一视觉层级，减少干扰，让用户专注主任务。

## 五、创新点
1. 目标澄清前置，避免模板化路径直接输出
2. 路径、任务、学习会话形成连续闭环
3. 引入学习状态指标，强化过程反馈
4. 前后端与 AI Gateway、Agent 模块形成完整系统
5. 既是作品展示，也是可运行的产品工程原型

## 六、技术方案
### 前端
- Vue 3 + TypeScript + Vite
- Pinia + Vue Router
- Element Plus + 自定义 `aa` 设计系统

### 后端
- Node.js + Express + TypeScript
- Prisma ORM
- JWT 鉴权
- 多业务路由 + 多服务模块
- AI Gateway 接入模型和 agent 系统

### 数据与部署
- 开发默认 SQLite
- 可扩展 PostgreSQL
- 支持 Nginx + Node 独立部署
- 通过 `.env` 管理模型、数据库和管理员配置

## 七、作品主要流程
### 用户主流程
1. 登录 / 注册
2. 进入学习总览
3. 发起目标对话
4. 自动生成学习路径
5. 查看路径详情与任务拆解
6. 进入对话学习或开放学习
7. 查看学习状态、成长成就与日志

### 管理员流程
1. 后台登录
2. 平台概览
3. 用户管理
4. 接口配置
5. 执行日志

- 把 AI 从答题工具升级为过程陪练和路径组织者
