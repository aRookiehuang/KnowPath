# 知途（KnowPath）用户手册

## 1. 适用对象
- 普通学习用户
- 平台管理员
- 测试与演示人员

## 2. 运行方式
### 2.1 本地启动
#### 启动后端
```bash
cd backend
npm install
npm run dev
```

#### 启动前端
```bash
cd frontend
npm install
npm run dev
```

#### Windows 一键启动
```powershell
./start-dev.ps1
```

### 2.2 默认地址
- 前端：`http://localhost:5173`
- 后端：`http://localhost:3001`
- 健康检查：`http://localhost:3001/health`

## 3. 账号说明
### 3.1 演示账号
- 用户名：`123`
- 密码：`123456`

### 3.2 管理员账号
管理员通常由后端环境变量初始化：
- `INIT_ADMIN_NAME`
- `INIT_ADMIN_PASSWORD`

管理员入口：`/admin/login`

## 4. 普通用户使用流程
### 4.1 登录
完成注册或登录后进入“学习总览”。

### 4.2 学习总览
可查看：
- 当前学习焦点
- 学习状态摘要
- 路径数量、任务进度、学习时长、等级等信息

### 4.3 目标对话
用于澄清：
- 当前想解决的真实问题
- 已有基础
- 时间限制
- 预期结果

建议按系统追问逐步回答，再生成学习路径。

### 4.4 学习路径
用户可以：
- 查看已生成路径
- 进入路径详情
- 查看阶段、任务和整体进度

### 4.5 任务详情
在任务详情页中，可以查看任务说明，并进入：
- 对话学习
- 开放学习

### 4.6 对话学习
对子任务进行训练式互动，AI 会提问、提示、反馈和推进。  
当前主链路已修复启动参数问题，正常入口会自动携带 `taskId` 与 `pathId`。

### 4.7 开放学习
适合自由探索、延展讨论和补充理解。

### 4.8 学习状态
可查看认知负荷、知识迁移、学习流、学习平衡等指标，用于辅助判断学习节奏。

### 4.9 成长成就
展示等级、经验、成就与成长里程碑。

### 4.10 个人资料 / 接口设置 / 运行日志
用户可在“个人与配置”区域查看资料、接口设置与运行日志。

## 5. 管理员使用流程
### 5.1 登录后台
访问 `/admin/login`，使用管理员账号登录。

### 5.2 平台概览
查看用户、路径、会话、智能体调用和近期活动情况。

### 5.3 用户管理
查看用户列表、状态和关键信息。

### 5.4 接口配置
查看并维护 AI 接口、模型与配置摘要。

### 5.5 执行日志
查看运行日志、异常事件与调用情况。

## 6. 常见问题
### 6.1 登录后页面异常
请先确认后端是否启动、接口地址是否正确，再检查浏览器控制台和后端终端输出。

### 6.2 对话学习提示缺少参数
请从路径详情或任务详情正常进入；当前主链路已修复该问题，若仍出现，应检查入口参数和后端接口。

### 6.3 为什么仍能看到 WenFlow 命名
这是历史命名遗留，不影响当前产品统一名称“知途（KnowPath）”。

## 7. 使用建议
- 先做目标对话，再生成路径
- 路径生成后优先进入具体任务而不是停留在概览层
- 遇到阻塞时，先用开放学习探索，再回到对话学习推进
- 定期查看学习状态和成长成就，调整节奏

## 8. 结语
知途的重点不是“替你回答”，而是“陪你推进”。建议围绕完整路径使用本系统，而不是仅把它当作聊天工具。
