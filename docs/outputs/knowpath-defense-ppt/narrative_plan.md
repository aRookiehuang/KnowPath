# Narrative Plan - KnowPath Defense PPT

## Audience
大学生计算机设计大赛“人工智能应用——实践赛道”评审老师。

## Objective
在 8~10 分钟内清晰说明：知途解决了什么真实问题、AI 在其中如何发挥作用、系统如何落地实现、作品为什么具备实践价值与推广潜力。

## Narrative Arc
1. 真实学习痛点
2. 项目定位与赛道契合
3. 解决方案与业务闭环
4. 核心功能与平台能力
5. 创新点与技术架构
6. 工程完成度与应用价值
7. 总结升华

## Visual System
- 16:9 横版
- 蓝紫科技教育风
- 浅色底 + 渐变装饰 + 圆角卡片
- 用产品截图、流程图、架构图、对比表承载主要信息
- 保持专业、清晰、适合答辩
