# 初始管理员账户配置

## 功能说明

系统会在首次启动时自动创建初始管理员账户，无需手动运行脚本。

## 配置方法

### 1. 编辑 backend/.env 文件

`ini
# ===========================================
# 初始管理员配置（仅首次启动时使用）
# ===========================================
INIT_ADMIN_NAME=admin
INIT_ADMIN_PASSWORD=YourSecurePassword123
`

### 2. 重启后端服务

`ash
cd backend
npm run dev
`

### 3. 查看日志

成功时会看到：
`
✅ 初始管理员创建成功：admin
`

如果已存在管理员：
`
✅ 已存在管理员账户，跳过创建
`

## 登录管理端

访问管理端登录页面，使用配置的用户名和密码登录：
- **用户名**: dmin（或你配置的值）
- **密码**: YourSecurePassword123（或你配置的值）

## 安全建议

1. ⚠️ **首次登录后立即修改密码**
2. ⚠️ **生产环境使用强密码**（至少 12 位，包含大小写字母、数字、特殊字符）
3. ⚠️ **不要将 .env 文件提交到 Git**（已自动忽略）

## 注意事项

- 管理员账户通过用户名识别（name 字段）
- 系统会自动创建邮箱：{用户名}@wenflow.local
- 如果已存在同名用户或管理员，会跳过创建
- 密码使用 bcrypt 加密存储（10 轮哈希）
